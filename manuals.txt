[ERROR]
Code: E101
Title: Motor Overheating
Description: The motor temperature exceeds the safe operating limit.
Cause: Cooling fan failure or blocked air intake.
Solution:
- Stop the machine immediately
- Wait at least 15 minutes for cooling
- Inspect and clean air vents
Safety:
- Do not touch the motor surface
- Wear heat-resistant gloves before inspection

[ERROR]
Code: E202
Title: Position Sensor Failure
Description: The position sensor is not providing valid readings.
Cause: Loose connection or damaged sensor cable.
Solution:
- Power off the machine
- Check sensor wiring and connectors
- Restart the system after inspection
Safety:
- Ensure power is fully disconnected before handling cables

[ERROR]
Code: E303
Title: Unexpected System Shutdown
Description: The machine shuts down without warning.
Cause: Power supply instability or internal protection trigger.
Solution:
- Verify power source stability
- Check system logs
- Contact maintenance if issue persists
Safety:
- Do not repeatedly restart the machine

[MAINTENANCE]
Task: Daily Inspection
Description: Visual inspection of the machine before operation.
Steps:
- Check for loose components
- Inspect cables and connectors
- Verify emergency stop button functionality
Frequency: Daily
Safety:
- Perform inspection before powering on the machine

[MAINTENANCE]
Task: Weekly Lubrication
Description: Lubrication of mechanical joints and moving parts.
Steps:
- Shut down the machine
- Apply approved lubricant to specified joints
- Remove excess lubricant
Frequency: Weekly
Safety:
- Wear protective gloves
- Use only manufacturer-approved lubricants

[SAFETY]
Topic: Emergency Stop Procedure
Description: How to safely stop the machine in an emergency.
Steps:
- Press the emergency stop button immediately
- Step away from the machine
- Inform the supervisor
Safety:
- Do not reset the emergency stop without authorization

[SAFETY]
Topic: Power Isolation
Description: Procedure for isolating power during maintenance.
Steps:
- Turn off the main power switch
- Lock the power source if possible
- Verify zero energy state
Safety:
- Never perform maintenance on an energized system

[OPERATION]
Task: Machine Startup Procedure
Description: Standard procedure for starting the machine.
Steps:
- Ensure all safety guards are in place
- Power on the control unit
- Start the machine from the control panel
Safety:
- Keep hands away from moving parts during startup

[OPERATION]
Task: Machine Shutdown Procedure
Description: Safe shutdown process after operation.
Steps:
- Stop active processes
- Shut down via control panel
- Turn off main power switch
Safety:
- Wait until all moving parts come to a complete stop
